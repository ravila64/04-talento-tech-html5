Etiquetas de edicion

En esta ocasion vamos a estudiar unas etiquetas de formato adicionales. Asi como existen las etiquetas de formato común, como <h1>, <h2> y <p>, existen otras que indican un formaro específico. A continuacion varias de las etiquetas que debes tener en cuenta:
Etiqueta Descripcion
<abbr> Para Abreviaciones
<b> En negrita
<blockquote> Referencia de un texto
<cite> Cita
<del> Texto eliminado - tachado
<em> Enfasis
<ins> Texto insertado
<meter> Medida
